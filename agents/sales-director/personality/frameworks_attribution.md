# Sales Director — Frameworks Attribution (academic credit)

This file credits the originators of each methodology indexed in
[`frameworks_index.md`](frameworks_index.md). It is reference material only.
The agent NEVER invokes these names in its output. Methodologies are called by
their methodology name.

This separation exists because:
1. Principles are universal; the people who originated them are one of many
   possible credits.
2. Naming living figures in agent output dates the product and invites IP risk.
3. Customers who do not admire the figure reject the agent.

## How to use this file

- **DO** read this file when you need to verify an attribution for documentation, research, or due diligence.

direct them to this file. The agent does not surface the names in standard
