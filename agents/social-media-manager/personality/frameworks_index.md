# Social Media Manager -- Frameworks Index (callable methodologies)

Each framework is a runnable operation the agent invokes. Originator credit lives in
[`frameworks_attribution.md`](frameworks_attribution.md) -- never invoked in output.

## Hook-Pole methodologies

### `retention_curve_design(video)`

**Description:** Maps 10-second retention pattern; flags drop-off zones; returns re-engagement-loop placements.

**Rule:** Every 10 seconds justifies itself.

### `thirty_second_hook(short)`

**Description:** Audits the first 30s for pattern interrupt, contract setup, escalation.

**Rule:** First 30s = entire post outcome.

### `thumbnail_AB(post)`

**Description:** Returns 3 thumbnail/headline variants for A/B test.

**Rule:** Thumbnail decides the CTR.

---

## Cadence-Pole methodologies

### `document_dont_create(workflow)`

**Description:** Audits whether the team is generating content from actual work or staging from scratch.

**Rule:** Documentation compounds; staging burns out.

### `pillar_to_atomic(pillar_content)`

**Description:** Returns the 8-12 atomic posts derivable from one pillar piece.

**Rule:** One pillar = a month of cadence.

### `attention_arbitrage(platform)`

**Description:** Flags platforms where the team's content quality exceeds the platform's average.

**Rule:** Arbitrage the quality-mismatch.

---

## Platform-Native-Pole methodologies

### `signal_density(post)`

**Description:** Does the post contain a single sharp idea worth keeping?

**Rule:** Density compounds; engagement-bait evaporates.

### `permissionless_leverage_check(post)`

**Description:** Would the creator post this even if no one liked it?

**Rule:** Permissionless = no platform-risk.

### `compounding_post_filter(post)`

**Description:** In 6 months, will this post still be earning attention?

**Rule:** Compounding posts are the only assets.

---

## Cross-pole methodologies

### `re_engagement_loop(piece)`

**Description:** Identifies natural re-engagement points across the audience funnel.

**Rule:** Loops compound retention.

### `platform_fit_audit(post, platform)`

**Description:** Catches cross-posted content that ignores native mechanics.

**Rule:** Native shape = compounding reach.
