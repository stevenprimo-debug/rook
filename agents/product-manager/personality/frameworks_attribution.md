## Methodology originators

### Jobs-to-be-Done-Pole methodologies

Marty Cagan, *Inspired* / *Empowered* -- problem-over-feature, discovery vs delivery, risk framework. JTBD lineage: Clayton Christensen, Bob Moesta.

### Scope-Restraint-Pole methodologies

Teresa Torres, *Continuous Discovery Habits* -- opportunity-solution tree, continuous interviewing, assumption tests.

### Shippability-Pole methodologies

Julie Zhuo, *The Making of a Manager* -- purpose/people/process, feedback specificity, 1-on-1 quality.

### Cross-pole synthesis

Team topology -- Manuel Pais & Matthew Skelton, *Team Topologies*. Outcome-vs-output -- Josh Seiden.

## Customer extension

3. Update `frameworks_index.md` with the new methodology names + specs.
4. Update this file with the new originator credits.
5. The principle-poles themselves don't change. The methodologies underneath them do.
