# SEO Specialist -- Frameworks Index (callable methodologies)

Each framework is a runnable operation the agent invokes. Originator credit lives in
[`frameworks_attribution.md`](frameworks_attribution.md) -- never invoked in output.

## SERP-Rank-Pole methodologies

### `skyscraper(query)`

**Description:** Finds current #1, returns the 10x version: longer, deeper, better-structured, better-sourced.

**Rule:** Build the page that deserves the rank.

### `content_upgrade(page)`

**Description:** Identifies the 3 missing-from-the-page elements that would push past current rank.

**Rule:** Upgrade beats rewrite.

### `keyword_intent_match(query, page)`

**Description:** Flags informational/transactional/navigational intent mismatch.

**Rule:** Intent mismatch = bounce.

### `pruning_audit(site)`

**Description:** Returns the bottom-20% pages to delete or merge.

**Rule:** Pruning lifts overall rank.

---

## Answer-Engine-Visibility-Pole methodologies

### `entity_disambiguation(brand, topic)`

**Description:** Audits whether answer engines can confidently identify the brand for this topic.

**Rule:** Disambiguation is the AEO substrate.

### `extractable_answer_check(page)`

**Description:** Can a 2-4 sentence answer be pulled from the page?

**Rule:** If no, the answer engine won't cite.

### `schema_markup_audit(page)`

**Description:** Returns missing schema types; recommends JSON-LD additions.

**Rule:** Schema = answer-engine readability.

### `citation_worthy_structure(page)`

**Description:** Audits whether the page is structured to be quotable.

**Rule:** Be quotable or be invisible.

---

## Topical-Authority-Pole methodologies

### `audience_fingerprint(market)`

**Description:** Returns specific audience composition.

**Rule:** Audience precedes cluster.

### `zero_click_strategy(topic)`

**Description:** For queries the user won't click -- how does the brand still win.

**Rule:** Win the SERP feature.

### `SEO_audit_SP2(site)`

**Description:** Strategic / prioritized / practical audit; returns ranked fix list.

**Rule:** Audit before either.

### `low_hanging_fruit_framework(site)`

**Description:** Identifies pages where rank-7-to-3 movement is cheapest.

**Rule:** Cheap wins compound credibility.

---

## Cross-pole methodologies

### `intl_SEO_targeting(market, hreflang)`

**Description:** Multi-region targeting audit.

**Rule:** hreflang or hurt-lang.

### `brand_lift_measurement(campaign)`

**Description:** Measures non-direct-attribution brand lift across SERP and AEO.

**Rule:** Brand lift > last-click attribution.
