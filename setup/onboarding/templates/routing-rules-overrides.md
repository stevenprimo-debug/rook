- **CRM tool name** → routes captures mentioning the customer's CRM to sales-director + prospecting-agent (the two agents that operate on pipeline data)

- **Competitor names** → routes competitor mentions to deep-researcher (intel job)

If the customer says "HubSpot" / "Salesforce" / etc. in any captured note, the inbox-routing skill will now route that capture to the right agent automatically.
