# PARA / BASB Reference Library — Canonical Index

This folder is the **single source of truth** for Tiago Forte's PARA Method and Building a Second Brain (BASB) reference material across the vault. All other depts cross-reference these files; only PERSONAL edits or adds.

## What PARA is

**P**rojects → **A**reas → **R**esources → **A**rchives. Four buckets, organized by **actionability** (not topic). Forte's framing for a digital second brain that survives across tools (Obsidian, Notion, Drive, etc.).

| Bucket | Definition | vault analog |
|---|---|---|
| **Projects** | Short-term efforts with a deadline + outcome | Active dispatch briefs, in-flight ASSIGN files |
| **Areas** | Long-term standards / responsibilities (no end date) | Agents themselves (sales-director, content-strategist, etc.) |
| **Resources** | Topics of ongoing interest, reference material | Per-dept `context/` + `reference/` folders |
| **Archives** | Inactive items from the other three | Closed deals, killed ideas, shipped projects |
