How does Higgsfield connect to AI agents?

Higgsfield uses MCP (Model Context Protocol), an open standard that gives AI agents access to external tools. Once connected, your agent can generate images, create videos, train characters, and browse your creation history, all within a single session.

Which agents are supported?

Higgsfield MCP works with Claude (web, Cowork, and Claude Code), OpenClaw, Hermes Agent, and NemoClaw. Any agent or client that supports MCP can connect to Higgsfield.

What models are available?

Higgsfield gives your agent access to 30+ models including Soul, Cinema Studio, Flux, Seedream, Kling, Minimax Hailuo, Veo, and more. Your agent automatically selects the best model for the task, or you can specify one yourself.

What can I create?

Images up to 4K resolution, videos up to 15 seconds across multiple cinematic styles, consistent characters using Soul training, and more. You can generate from text prompts, reference images, or a combination of both.

Do I need an API key?

No. Add the Higgsfield MCP server URL in your agent's settings and authenticate through your Higgsfield account. No API keys to manage or configure.

How does pricing work?

Higgsfield tools use the same credit system as the Higgsfield platform. Each generation costs credits based on the model and resolution. Your existing Higgsfield plan credits work seamlessly through any connected agent.

How long does generation take?

Images typically complete in a few seconds. Videos take longer depending on duration and model. All generation runs asynchronously, so your agent polls for results and delivers them as soon as they're ready.

Can I use previous generations as input?

Yes. You can browse your full generation history, reference any past image or video, and use it as a starting point for new creations. This makes iterative workflows and multi-step production easy.

Can I use Higgsfield with my own agent setup?

Yes. Any MCP-compatible client can connect. If you're running OpenClaw on a VPS, Hermes Agent on your local machine, or NemoClaw on NVIDIA hardware, just point it to the Higgsfield MCP server and you're ready to generate.
